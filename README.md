# 股票智能分析系统 API

## 概述
基于多维度技术指标的智能股票评分系统

## 评分规则（满分100分）

### 1. 均线系统 (+25分)
- 均线多头排列 (MA5>MA10>MA20): +25分
- 均线死叉: -15分
- 均线空头排列: -20分

### 2. WR威廉指标 (+15分)
- WR≤20 (超卖反弹): +15分
- WR≥80 (超买风险): -10分

### 3. 资金流向 (+20分)
- 资金流入 (放量>1.3倍+上涨): +20分
- 资金流出 (放量>1.5倍+下跌>2%): -15分

### 4. 位置判断 (+15分)
- 接近新高 (95%+): +15分
- 超跌反弹 (接近新低+资金流入): +10分

### 5. 涨跌幅 (+10分)
- 跌幅较大+资金流入 (超跌反弹): +10分
- 涨幅较大: -5分

### 6. 连续性 (+5分)
- 连续下跌+资金流入: +10分
- 连续上涨: -5分

## 信号评级

| 分数 | 信号 | 建议 |
|------|------|------|
| ≥35 | 强烈买入 | 积极建仓 |
| ≥20 | 买入 | 适量买入 |
| ≥10 | 加仓 | 可考虑加仓 |
| ≥0 | 观望 | 保持关注 |
| ≥-10 | 减仓 | 建议减仓 |
| <-10 | 清仓 | 建议清仓 |

## API接口

### 单股分析
```
GET /api/analyze/<股票代码>
```

### TOP股票推荐
```
GET /api/top-stocks
```

### 涨停模式分析
```
GET /api/limitup-pattern
```

### 全盘扫描
```
GET /api/scan-all
```

## 技术栈
- 后端: Flask + Vercel
- 数据源: 东方财富API
